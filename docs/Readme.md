# **Prueba de Validación Multiproceso**

## _Ejercicio 1_
___

- Este programa generará y matará 10 procesos y mostrará información sobre ellos.
- Consta de una función padre que crea 10 procesos, uno cada 10 segundos, mostrando el PID junto a la hora de inicio de cada proceso generado.
- También tiene una función hijo que mostrará cada vez que se genere un proceso un mensaje de inicio con su PID y a los 3 segundos otro mensaje de finalización junto al PID.


> **->** _Este programa sólo puede ser ejecutado en máquinas Linux o Mac OS._
